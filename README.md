## Команды / функции
## pathes
### pathes.getPath(line: str)
Основная функция для обработки входщих строк. 

* **Входные данные**: команда, записанная пользователем в терминале;
* **Вывод**: словарь вида:

```
dict = {
    'com' : <Команда для выполнения: str>,
    'opt': <Список переданных опций: list>,
    'path': <Список переданных адресов: list>,
}
```

**False** - если был передан несуществующий путь.

### pathes.getPathWithSpace(line: str)

Обрабатывает строку, если в ней есть названия файлов/директорий с пробелами. Возвращаемые данные аналогичны **pathes.getPath()**.


### pathes.CheckRightPath(path: str)

Проверяет существует ли переданные путь.

* **Входные данные**: путь;
* **Вывод**:  
    * **True** - если путь существует;
    * **False** - если путь не существует (через декоратор **@log.get_mistake**)

### pathes.getOption(line: str)
Выделяет опции из строки.

* **Входные данные**: строка;
* **Вывод**:
    * **option: str** - если формат записи опции был обнаружен;
    * **None** - если формата записи опции не было обнаружено.

-----
В функциях выполнения команд не требуется проверки на правильность переданного пути, потому что **pathes.getPath()** отсеивать несуществующие пути.

## log
### log.make_log()
Создает файл для логгирования или определяет уже существующий. Определяет формат записи логов.

### log.log_in(line: str)
Записывает переданную строку в лог.

### log.get_mistake(func)
Функция для декоратора **log.get_mistake**. На вход принимает возвращаемое значение (return) из обёртываемой функции (**func**). 
* Если **func** выполняется исправно и возвращает **True** - не изменяет поведение функции. 
* В противном случае выводит сообщение об ошибке (передаётся обёртываемыми функция в **return** в случае возникновения ошибки) и передаёт **ERROR**-сообщение в функцию **log.log_in()** для записи отчёта об ошибке. Возращает **False** для выхода из обёртываемой функции и завершения её исполнения.

## ls
### ls.ls(PathList: list, OptionList: list)
Отображает список файлов и каталогов в переданном каталоге.

Поддерживает **PathList = [ ]** --> будет отображено содержание текущего рабочего каталога.

Поддерживает множественную передачу путей в **PathList** --> отображение будет выводиться поочерёдно.

При передаче файла в **PathList** --> выведет переданный путь.

Поддерживается только опция **-l** для отображения подробной информации: *имя, рамер, дата изменения, права доступа*.
При передачи опции, отличной от **-l** - завершение работы команды, вывод сообщения: **ls: invalid option -- "<переданная опция>"** и запись сообщения об ошибке в лог.
